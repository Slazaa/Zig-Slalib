# Slalib
A anternative for the standard library inspired from the Rust standard library. For now, Slalib uses libc but in the future it will not depend on anything.

## Goals
- [ ] Collections
	- [ ] Queue
	- [ ] Map
	- [x] Vec
- [ ] Format
- [ ] I/O
	- [ ] File
	- [ ] Stdio
- [x] Math
- [x] Memory
	- [x] Allocator
	- [x] GlobAlloc
- [ ] Network
	- [ ] Tcp
	- [ ] Udp
- [x] String
- [ ] Thread
- [ ] Time
